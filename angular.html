Définition

Angular est un framework ( Un framework logiciel est un ensemble d'outils et de composants à partir desquels on peut développer des logiciels ) Javascript développé par Google. Il est utilisé pour réaliser des applications web et/ou mobile. Il offre la possiblité de réaliser des interfaces SPA ( Single Page Application ) qui ne nécessite pas de rechargement. Angular est un language open source qui se voit régulièrement enrichi et alimenté par enormement de développeurs.

Angular s’appuie sur 3 languages principalement :

l’HTML qui va servir à la structure de l’application
le SCSS ou CSS qui va permettre d’embellir notre solution logicielle
 le TypeScript pour tout ce qui est dynamique, comportement et données
Angular est un framework complet – on peut créer des applications web complètes sans avoir besoin de librairies tierces supplémentaires. C'est notamment ce qui différencie un framework d'une librarie.

Apprentissage

J'ai découvert le framework Angular lors de mon cursus d'ingénieur logicel à IN'TECH. J'ai réussis à valider ce module assez facielement, je m'entrainnais régulierement en réalisant diverses application web très simples. J'ai notemment utilisé le site FrontEnd Mentor afin d'améliorer mes compétences en Angular.
J'ai réalisé une application extrement simple pour afficher une citation récupérée via une API : Advice Generator Puis j'ai réalisé une application plus conséquante au niveau du FRONT : Invoice App. Cette application n'est pas terminée, elle nécessiterai d'être connecté à une API, on ne peut pas ajouter ou supprimer de facture.
Si j'ai décidé de ne pas la continuer c'est en partie car, ce que je visais réelement avec ce projet était de pouvoir suivre une maquette professioinelle ( fournie par FrontEnd Mentor ). J'ai selon moi réussis ce que je voulais faire, je suis donc passé à autre chose.

J'ai ensuite utilisé quasi-quotidiennement Angular lors de mon alternance chez Sigems - Berger Levrault, sur le projet BL.PMSI. J'ai pu encore m'ameliorer, et j'y ai appris à utilser le framework Tailwind afin d'alleger mes classes CSS et surtout mettre en place assz facilement le responsive. J'ai aussi pu utilser le framework en combinaise avec le framework Ionic, utilisé pour creer des application mobiles notamment. J'ai pu l'utiliser pour réaliser cette application : Vox Populi, disponible sur play store.

Cette compétence me permet de completer mon profil full stack, afin de pouvoir agir sur toute la chaine. Elle me permet aussi de pouvoir avoir un oeil de connaisseur sur l'architecure, déployement d'une application Front.
