Définition

Angular est un framework (Un framework logiciel est un ensemble d'outils et de composants à partir desquels on peut développer des logiciels) Javascript développé par Google. Il est utilisé pour réaliser des applications web et/ou mobile. Il offre la possiblité de réaliser des interfaces SPA ( Single Page Application ) qui ne nécessite pas de rechargement. Angular est un language open source qui se voit régulièrement enrichi et alimenté par enormement de développeurs.

Angular s’appuie sur 3 languages principalement :
l’HTML qui va servir à la structure de l’application
le SCSS ou CSS qui va permettre d’embellir notre solution logicielle
 le TypeScript pour tout ce qui est dynamique, comportement et données
Angular est un framework complet – on peut créer des applications web complètes sans avoir besoin de librairies tierces supplémentaires. C'est notamment ce qui différencie un framework d'une library
