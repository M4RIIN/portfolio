Angular

LIEN

Définition

Angular est un framework (Un framework logiciel est un ensemble d'outils et de composants à partir desquels on peut développer des logiciels) Javascript développé par Google. Il est utilisé pour réaliser des applications web et/ou mobile. Il offre la possiblité de réaliser des interfaces SPA (Single Page Application) qui ne nécessitent pas de rechargement. Angular est un language open source qui se voit régulièrement enrichi et alimenté par énormement de développeurs.

Angular s’appuie sur 3 langages principalement :

l’HTML qui va servir à la structure de l’application,
le SCSS ou CSS qui va permettre d’embellir notre solution logicielle,
 le TypeScript pour tout ce qui est dynamique, comportement et données
Angular est un framework complet – on peut créer des applications web complètes sans avoir besoin de librairie tierces supplémentaires. C'est notamment ce qui différencie un framework d'une librarie.

Apprentissage

J'ai découvert le framework Angular lors de mon cursus d'ingénieur logicel à IN'TECH Sud Agen. J'ai réussi à valider ce module assez facilement, je m'entraînais régulièrement en réalisant diverses applications web très simples. J'ai notamment utilisé le site FrontEnd Mentor afin d'améliorer mes compétences en Angular.
J'ai réalisé une application extrêmement simple pour afficher une citation récupérée via une API : Advice Generator. Puis j'ai réalisé une application plus conséquente au niveau du FRONT : Invoice App. Cette application n'est pas terminée, elle nécessiterait d'être connectée à une API, on ne peut pas ajouter ou supprimer de facture.
Si j'ai décidé de ne pas la continuer c'est en partie car, ce que je visais réellement avec ce projet était de pouvoir suivre une maquette professionelle (fournie par FrontEnd Mentor). J'ai selon moi réussi ce que je voulais faire, je suis donc passé à autre chose.

J'ai ensuite utilisé quasi-quotidiennement Angular lors de mon alternance chez Sigems - Berger Levrault, sur le projet BL.PMSI. J'ai pu encore m'améliorer, et j'y ai appris à utiliser le framework Tailwind afin d'alléger mes classes CSS et surtout mettre en place assez facilement le responsive. J'ai aussi pu utiliser ce framework en combinaison avec le framework Ionic, utilisé pour créer des applications mobiles notamment. J'ai pu l'utiliser pour réaliser cette application : Vox Populi, disponible sur play store.

Autocritique

Cette compétence me permet de compléter mon profil full stack, afin de pouvoir agir sur toute la chaine du BACK-END jusqu'au FRONT-END. Elle me permet aussi de pouvoir avoir des connaissances solides sur l'architecture et le déploiement d'une application Front. Il m'a fallu du temps pour acquerir les compétences suffisantes mais je pense maintenant avoir une bonne maîtrise d'Angular pour réaliser des outils web. Ayant un profil orienté back-end, j'ai toujours eu un peu de mal à créer quelque chose au design "irréprochable", pendant longtemps j'ai bloqué sur le CSS, je conseille donc de se focaliser sur la fluidité et l'expérience utilisateur que peut apporter Angular.
