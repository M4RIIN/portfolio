Définition

Java est un langage de programmation orienté objet. Il permet de créer des applications complètes. Il s'agit d'un langage dit "compilé", il nécessite d'une JVM ( Java Virtual Machine ) ce qui lui permet d'être executer sur n'importe quelle platforme. La machine virtuelle Java comprend un compilateur JIT (Just-In-Time), ou compilateur à la volée qui compile dynamiquement le code source en code exécutable au lieu de l'interpréter instruction par instruction.

Apprentissage

J'ai découvert et appris à utiliser Java lors de mon cursur d'ingenieur logiciel. J'y ai appris les bases ainsi que les bases de la programamtion orienté objet. J'ai ensuite au l'occasion de l'utiliser quotidiennement lors de mon alternance chez Sigmes / Berfer-Levrault. Le framework SpringBoot est basé sur Java, que j'ai donc utilisé tout deux lors de mon alternance. Java est extrement populaire et fait partie des lanagages les plus populaire en 2022.

Java était donc au coeur du développement du projet BL.PMSI il nous a donc permis de creer ce projet et de le rendre fonctionnel. Apres plus de 3 ans d'utilisation presque quotidienne, je sais utiliser Java dans un environnement complexe comme on eput en trouver en entreprise. J'ai aussi appris à compiler et déployer une application Java sur un server. Je pense avoir un très bonne connaissance de ce langage.

Néamoins, Java est un language qui évolue très rapidement, nous en somme actuellement à la version 18. De nombreuse fonctionnalités sont régulierment ajoutées ( lamba expression, modules etc..) il me reste donc beaucoup à apprendre dans le futur. J'aime énorment ce langage car il permet de trvailelr sur enorment de type de projet differents. Utiliser conjoitnement avec Maven et Jenkins, il permet de creer une application répondant aux besoins d'enroement de cleint et d'assurer une sécurité lors des mise à jour, grâce à la CI/CD.

Java me permet donc d'être présent sur la marché de création d'application en client lourd, en API. Il est ausis la base de beaucoup de framework ( SpringBoot ) et d'autres langauge se base sur lui, comme Java Andorid, qui permet de creer des application mobile pour Android. Afin de pouvoir prendre en main convenablement Java, il est nécessaire d'apprendre les principes de conception orineté objet.

Ce langage m'as permis d'approndire mes connaissance en architecure logicielle, car j'ai pu creer bon nombnre de petit projet afin d'apprendre les bases de la "clean architecure". Associé à Maven il permet de découper sont code et de reduire les dépendance forte entre nos modules.
