Définition

Java est un langage de programmation orienté objet. Il permet de créer des applications complètes. Il s'agit d'un langage dit "compilé", il nécessite une JVM ( Java Virtual Machine ) ce qui lui permet d'être exécuté sur n'importe quelle plateforme. La machine virtuelle Java comprend un compilateur JIT (Just-In-Time), ou compilateur à la volée qui compile dynamiquement le code source en code exécutable au lieu de l'interpréter instruction par instruction.

Apprentissage

J'ai découvert et appris à utiliser Java lors de mon cursus d'ingénieur logiciel. J'y ai appris les bases ainsi que les bases de la programmation orienté objet. J'ai ensuite eu l'occasion de l'utiliser quotidiennement lors de mon alternance chez Sigems / Berger-Levrault. Le framework SpringBoot est basé sur Java, que j'ai donc utilisé tous deux lors de mon alternance. Java est extrêmement populaire et fait parti des langages les plus populaires en 2022.

Java était donc au cœur du développement du projet BL.PMSI il nous a donc permis de créer ce projet et de le rendre fonctionnel. Après plus de 3 ans d'utilisation presque quotidienne, je sais utiliser Java dans un environnement complexe comme on peut en trouver en entreprise. J'ai aussi appris à compiler et à déployer une application Java sur un serveur. Je pense avoir une très bonne connaissance de ce langage.

Néanmoins, Java est un langage qui évolue très rapidement, nous en sommes actuellement à la version 18. De nombreuses fonctionnalités sont régulièrement ajoutées (lambda expression, modules etc..) il me reste donc beaucoup à apprendre dans le futur. J'aime énormément ce langage car il permet de travailler sur un grand nombre de types de projet différents. Utilisé conjointement avec Maven et Jenkins, il permet de créer une application répondant aux besoins de la plupart des clients et d'assurer une sécurité lors d'une mise à jour, grâce à la CI/CD.

Java me permet donc d'être présent sur le marché de création d'application en client lourd, en API. Il est aussi la base de beaucoup de framework (SpringBoot) et d'autres langages se basent sur lui, comme Java Android, Kotlin qui permet de créer des applications mobiles pour Android par exemple. Afin de pouvoir prendre en main convenablement Java, il est nécessaire d'apprendre les principes de conception orientée objet.

Ce langage m'a permis d'approfondir mes connaissances en architecure logicielle, car j'ai pu créer bon nombre de petits projets afin d'apprendre les bases de la "clean architecure". Associé à Maven il permet de découper son code et de réduire les dépendances fortes entre nos modules.
