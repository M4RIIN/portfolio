C#

LIEN

Définition

C# est un langage de programmation orienté objet, dérivé de C et de C++. Le C# permet de développer aussi bien des applications web (via l'ASP .NET) que des clients lourds (console, winform,etc) ou encore des librairies de fonctions (DLL).

Apprentissage

J'ai découvert et appris à utiliser le C# lors de mon cursus à IN'TECH Sud AGEN. Lors d'un projet de réalisation d'un calendrier spécifique pour UPSA. C# nous a permis de connecter très facilement notre gestionnaire de planning à Microsoft et sa suite d'outils (ex. Outlook).

Nous avons eu des cours de C# pendant plusieurs mois en plus du projet qui a duré 6 mois. Et notamment sur un framework : Xamarin pour générer des applications hybrides iOs et Android.

En plus de cela le projet que j'ai réalisé : LiveSplit Information est réalisé en C#, car le logiciel LiveSplit est lui aussi créé en C#. J'ai donc une bonne base de connaissance sur ce langage.

Néanmoins C# est un langage qui évolue très vite, son environnement aussi d'ailleurs, il me reste donc énormément de choses à apprendre sur ce langage et ses possibilités. Cependant, je sais réaliser une application de bureau grâce au WPF, une API et une solution web grâce à .NET.

Autocritique

Ce langage m'a permis d'approfondir mes connaissances en architecture logicielle, car j'ai pu créer bons nombres de petits projets afin d'apprendre les bases de la "clean architecure". Associé au gestionnaire de package NuGet il permet de découper son code et de réduire les dépendances fortes entre nos packages.

C# est langage que j'apprécie beaucoup, j'aimerais continuer de l'utiliser dans le futur afin de monter en compétences sur ce langage dont je pense avoir un niveau satisfaisant.

Je pense qu'il est nécessaire d'avoir de solides connaissances sur la programmation orientée objet avant de pouvoir se lancer dans le C# afin de pouvoir proposer du code clair, concis et qui suit les principes SOLID. Cette compétence est importante dans mon parcours d'ingénieur, car c'est un des langages phares du développement logiciel. J'aimerais pouvoir maîtriser cette compétence car elle est très recherchée. Pour parfaire mes connaissances sur le C# je vais continuer de l'utiliser pour mettre à jour le projet : LiveSplit Information.
