C#

LIEN

Définition

C# est un langage de programmation orientée objet, dérivé de C et de C++. Le C# permet de développer aussi bien des applications web (via l'ASP .NET) que des clients lourds (console, winform,etc), des librairies de fonctions (DLL).

Apprentissage

J'ai découvert et appris à utiliser le C# lors de mon cursur à Intech Sud AGEN. Lors d'un projet de réalisation d'un calendrier spécifique pour UPSA. C# nous a permis de connecter très facilement notre gestionnaire de planning à la Mircosoft et sa suite d'outils ( ex. Outlook ).

Nous avons eu des cours de C# pendant plusieur mois en plus du projet qui à duré 6 mois. Et notamment sur un framework : Xamarin pour generer des application hybride iOs, Android.

En plus de cela le projet que j'ai réalisé : liveSplit Information est réalisé en C# car le logiciel LiveSplit est lui aussi creer en C#. J'ai donc une bonne base de connaissance sur ce langage.

Néamoins C# est un langage qui évolue très vite, son enviroonement aussi d'ailleurs il me reste donc enromeemnt de chose à apprendre sur ce langage et ces possibilités. Cependant je sais réaslier une application de bureau grâce au WPF , une API et une solution web grace à Razor.

Autocritique

Ce langage m'as permis d'approndire mes connaissance en architecure logicielle, car j'ai pu creer bon nombnre de petit projet afin d'apprendre les bases de la "clean architecure". Associé au gestionnaire de package NuGet il permet de découper sont code et de reduire les dépendance forte entre nos packages.

C# est langage que j'apprecie beaucoup, j'aimerai contiunuer de l'uiliser dans le futur afin de monter en compétence sur ce langage dont je pense avoir un niveau satisfaisant.

Je pense qu'il est nécessaire d'avoir des solides connaissance sur l'orienté objet avant de pouvoir se lancer dans le C# afin de pouvoir proser du code clair, concis et qui suit les principes SOLID. Cette compétence est important dans mon parcours d'ingenieur car c'est un des langages phare du developpement logiciel. J'aimerai pouvoir maîtrisé cette compétence car elle est très recherché dans ce domaine. Pour parfaire mes connaisance sur le C# je vais continuer de l'utiliser pour mettre à jour le projet : LiveSplit Information.
